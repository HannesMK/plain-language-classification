Die Nasennebenhöhlen können sich entzünden. 
Dann ist oft die Nase verstopft. 
Eine Nasennebenhöhlen-Entzündung kann ansteckend sein. 
Aber ein Arzt kann die Krankheit behandeln.
Die Nasennebenhöhlen sind nahe der Nase. 
Die Nasennebenhöhlen sind Hohlräume. 
Sie sind mit Luft gefüllt. 
In den Hohlräumen ist eine Schleimhaut. 
Die Schleimhaut befeuchtet die Nebenhöhlen mit einem Sekret. 
So schützt die Schleimhaut die Nebenhöhlen vor Krankheitserregern. 
Trotzdem können Bakterien oder Viren die Schleimhaut reizen. 
Dann produziert die Schleimhaut zu viel Sekret und schwillt an.
Eine Nasennebenhöhlen-Entzündung kann akut sein. 
Dann dauert sie ungefähr zwei bis vier Wochen.
Aber eine Nasennebenhöhlen-Entzündung kann auch chronisch werden. 
Chronisch bedeutet: 
Die Entzündung entwickelt sich langsam und bleibt oft sehr lange Zeit. 
Eine chronische Nasennebenhöhlen-Entzündung kann länger als zwölf Wochen dauern.
Die Ursachen einer akuten Nasennebenhöhlen-Entzündung können sein:
Erkältung Heuschnupfen Zahnprobleme
Eine Erkältung ist die häufigste Ursache von einer akuten Nasennebenhöhlen-Entzündung.
Die Ursachen einer chronischen Nasennebenhöhlen-Entzündung können sein:
Nasenpolypen Asthma ein schwaches Immunsystem
Eine akute Nasennebenhöhlen-Entzündung kann verschiedene Anzeichen haben. 
Zum Beispiel:
Kopfschmerzen eine verstopfte Nase Zahnschmerzen im Oberkiefer ein starkes Krankheitsgefühl
Die Nasennebenhöhlen-Entzündung ist chronisch ? 
Dann sind die Anzeichen nicht so leicht zu erkennen.
Sie haben Kopfschmerzen, bekommen schlecht Luft durch die Nase oder können nicht mehr so gut riechen?
Dann können das Anzeichen für eine chronische Nasennebenhöhlen-Entzündung sein.
Sie glauben: 
Ich habe vielleicht eine Nasennebenhöhlen-Entzündung? 
Dann gehen Sie zum Arzt. 
Der Arzt untersucht Sie und kann Ihnen helfen.
Sprechen Sie mit einem Arzt. 
Der Arzt gibt Ihnen alle wichtigen Informationen. 
Bei einer akuten Nasennebenhöhlen-Entzündung helfen:
Nasenspülungen mit Salzlösungen Inhalieren von Wasserdampf Schmerzmittel Nasenspray
Bei einer chronischen Nasennebenhöhlen-Entzündung kann der Arzt Nasensprays mit Kortison verschreiben. 
Kortison ist ein Medikament. 
Durch das Kortison schwillt die Schleimhaut schneller ab. 
So kann das Sekret besser ablaufen. 
Bei einer chronischen Nasennebenhöhlen-Entzündung helfen auch Nasenspülungen mit Salzlösungen.
Jeder Mensch kann selbst etwas gegen eine Nasennebenhöhlen-Entzündung tun. 
Zum Beispiel:
nicht rauchen gesunde Ernährung Bewegung ausreichend Schlaf richtig Schnäuzen: 
Schnäuzen Sie mit möglichst wenig Druck. 
Putzen Sie immer beide Nasenhälften getrennt: 
Halten Sie ein Nasenloch zu und blasen Sie durch das andere. 
Dann umgekehrt.
